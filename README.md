== Description ==
Asesor Inteligente es un plugin gratuito que agrega un asistente digital básico a tu sitio WordPress.
Ideal para validar ideas, recibir feedback y captar leads iniciales.

== Installation ==
1. Sube el plugin desde Plugins > Añadir nuevo > Subir plugin
2. Actívalo
3. Listo
